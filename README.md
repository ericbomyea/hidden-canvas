# Hidden Canvas

A calming puzzle experience: word search meets paint-by-numbers. Tap numbers to solve a quick word search, then tap cells to paint.

- **Stack:** React / Next.js (web app, mobile-responsive)
- **Deploy to Vercel:** From this directory run `npx vercel` (log in with `npx vercel login` if needed). Or connect this repo at [vercel.com/new](https://vercel.com/new).
